# OS_project1
NTU 2020 Spring, Operating System

## 執行環境
 + linux version --> 4.14.25
 + Ubuntu --> 16.04
 
## 執行方式
 + Compile
     ```bash
     make
     ```
 + 執行 scheduler
     ```bash
     ./scheduler < [input file]
     ```
 + 若需要重現 output，則需要將 OS_PJ1_Test 放至目錄底下並執行
      ```bash
     bash gen_output.sh
     ```
